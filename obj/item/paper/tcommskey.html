Telecommunications Password Paper
/obj/item/paper/tcommskey

Piece of paper that spawns with the default link password

This is spawned in the CE office and has the default link password While convenient, this is not necessary and doesnt matter if it gets lost or destroyed Because you can view the password easily by just looking at the core link page
